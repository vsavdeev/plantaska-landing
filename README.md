# Plantaska Landing

Статический лендинг для мобильного приложения **Plantaska** (mobile‑first).

## Как открыть

- **Вариант 1 (быстро):** откройте файл `index.html` в браузере.
- **Вариант 2 (правильно):** поднимите простой сервер в папке проекта и откройте страницу:

```bash
cd /Users/vavdeev/Documents/plantaska-landing
python3 -m http.server 5173
```

Затем перейдите в браузере на `http://localhost:5173`.
